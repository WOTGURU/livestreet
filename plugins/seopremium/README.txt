ОПИСАНИЕ
--------

Основное предназначение плагина это оптимизация структуры сайта на LiveStreet CMS
для поисковых систем. Плагин позволяет добавлять собственные значения мета тегов
keyword и description. По умолчанию значения берутся из названия и описания блогов и топиков
соответственно. Также существует возможность собственные значения для каждого конкретного
блога или топика, которое будет выводится на соответствующей странице сайта.
Для блогов - страница со списком топиков.
Для топика - страница просмотра топика.
Данная возможность позволит более гибко управлять SEO данными на страницах сайта. 
Присутствует интеграция с плагином l10n.

ИСТОРИЯ ВЕРСИЙ
--------------

v0.1
- Реализована возможность добавления собственных значений  полей заголовка meta: "keyword" "description",
задавать которые можно на странице редактирования топика - для топика, и блога - для блога соответственно.
(В случае если значения не заданы, будут выводится данные сгенерированные модулем SEO)



Внимание! Плагин может работать только совместно с плагином SEO версии не ниже 0.3.0
В ином случае, корректная работа плагина не гарантируется.
В случае работы плагина совместно с плагином L10n версия плагина (L10n) должа быть не ниже 0.4.1
