Плагин AutoOpenId (версия 1.5.44) для LiveStreet 1.0

Описание
Позволяет пользователям авторизоваться/зарегистрироваться через ВКонтакте, Facebook, Twitter, Gmail, Yandex, Rambler, или любого другого провайдера OpenId (включая LiveJournal).
Автоматически загружает логин (все провайдеры), аватарку и пол пользователя (facebook, twitter, vKontakte). Пользователь регистрируется на сайте за 1 клик.
Есть возможность привязки OpenID к уже существующим пользователям.
Плагин добавляет форму авторизации через openId на стреницы http://<ваш_сайт>/login и http://<ваш_сайт>/registration/

Для работы должны быть подключены библиотеки Curl и json.
Если плагин будет работать вместе с плагином Reposter раскомментируйте строчки в конфиг файле, соответствующие запросу расширенных прав у пользователя вашими приложениями соц. сетей (Facebook и ВКонтакте).



УСТАНОВКА
1. Скопировать плагин в каталог /plugins/
2. Через панель управления плагинами (/admin/plugins/) запустить его активацию. В панели управления он будет называться AutoOpenId.
3. Выполнить настройки плагина в файле /plugins/autoopenid/config/config.php,

Для авторизации через ВКонтакте необходимо создать standalone приложение на сайте vkontakte.ru (http://vkontakte.ru/editapp?act=create&site=1) и прописать его параметры в конфиге.
Более подробно о ВКонтакте можно прочитать здесь - http://vkontakte.ru/developers.php?o=-1&p=Open+API 

Для Facebook также необходимо указать параметры приложения в конфиге. Регистрация и управление приложениями осуществляется со страницы  https://developers.facebook.com/apps/ 

Для Twitter: регистрация приложения -- http://dev.twitter.com/apps/new, подробнее здесь - http://dev.twitter.com/pages/auth . В поле Callback Url укажите адрес Вашего сайта.
Проверьте крректность заполнения поля Callback url. В нем должен быть адрес Вашего сайта ( 	http://site.ru/ )
Для публикации сообщений от имени пользователя при помощи плагина Reposter выбирайте тип доступа read and write



ОБНОВЛЕНИЕ ПРЕЖНЕЙ ВЕРСИИ 
0. Деактивировать плагин.
1. Распаковать архив.
2. Заполнить config.php новой версии.
3. Переименовать папку со старой версией в каталоге ЛС.
4. Скопировать папку autoopenid с новой версией в каталог плагинов ЛС.
5. Активировать плагин.


Тестировать плагин после установки лучше на странице http://<ваш_сайт>/login/ , так  как на этой странице обычно нет дополнительных скриптов.

Плагин "OpenID" (версия 1.2.0) для LiveStreet 0.4.2

ОПИСАНИЕ
Позволяет пользователям авторизоваться/зарегистрироваться через OpenID, ВКонтакте, Facebook, Twitter, Gmail, Yandex, Rambler.
Есть возможность привязки OpenID к уже существующим пользователям.
Плагин добавляет кнопки OpenID входа на страницы регистрации и авторизации.

УСТАНОВКА
1. Скопировать плагин в каталог /plugins/
2. Через панель управления плагинами (/admin/plugins/) запустить его активацию.
3. Выполнить настройки плагина в файле /plugins/openid/config/config.php, для авторизации через ВКонтакте необходимо создать приложение на сайте vkontakte.ru и прописать его параметры в конфиге.
	Более подробно о ВКонтакте можно прочитать здесь - http://vkontakte.ru/developers.php?o=-1&p=Open+API
	Для Facebook также необходимо указать параметры в конфиге, подробнее здесь - http://developers.facebook.com/docs/guides/canvas/
	Для Twitter подробнее здесь - http://dev.twitter.com/pages/auth
